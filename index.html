Sección 1: Introducción

Este es un ejemplo de página web que sigue los principios de accesibilidad. El texto debe tener suficiente contraste con el fondo y proporcionar descripciones de texto alternativas para las imágenes, ¡son prácticas importantes!

Sección 2: Contenido Adicional

Aquí hay más contenido para ilustrar cómo estructurar una página web accesible. Las etiquetas semánticas como <nav> y <main> ayudan a organizar el contenido de manera clara.

Sección 3: Enlaces y Botones

Se han utilizado colores y contrastes adecuados para los enlaces y botones, facilitando la identificación y la navegación para todos los usuarios.
